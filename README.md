# nestjs-api
This is a backend CRUD API built using **NestJS**, **TypeORM**, **JWT Authentication**, and tested with **Postman**.  
The project demonstrates clean architecture, modular structure, and secure authentication flows.



## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
